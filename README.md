# chatbot-Transformer-&-Linformer

## Main requirements

- Python 3.6 to 3.8*
- torch==1.7.1 (PyTorch)
- torchtext==0.4.0
- nltk==3.4.5 (Natural Language Toolkit)

*Require for many dependencies in requirement.txt
.
## How to Start

- install dependencies

`$ pip install -r requirements.txt`
